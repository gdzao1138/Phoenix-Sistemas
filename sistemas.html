AGENDAMENTO:

Passo 1:

O usuario recebe o aplicativo juntamente com um "e-mail e "senha" para ter acesso ao programa

Passo 2:

O usuario terá acesso ao programa, onde o mesmo terá que preencher os dados de acordo com o que for informado do cliente, a ponto que crie uma aba com todos os dados

Passo 3:

Após ser adicionado os dados será criado uma aba, onde o usuario poderá editar, deletar ou marcar o serviço como concluido


CONTROLE DE TAREFAS:


Passo 1:

O usuario iniciará colocando uma primeira tarefa

Passo 2:

Após ser adicionado a tarefa, será criado uma aba com a tarefa

Passo 3:

O usuario poderá realizar seguintes ações com aba: edita-la caso aja alguma alteração, deleta-la caso seja necessario, colocar em checkbox que a atividade se encontra concluida
